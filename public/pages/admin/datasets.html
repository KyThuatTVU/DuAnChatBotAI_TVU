📝 Định dạng file Word

Mỗi câu hỏi bắt đầu bằng "Câu hỏi:" hoặc "Q:". Câu trả lời tiếp theo bắt đầu bằng "Câu trả lời:" hoặc "A:".

Tương tự Word. Hệ thống sẽ trích xuất nội dung text từ PDF, nên tránh file scan ảnh.
